Segment tree to hold nodes (segments) of intervals.

Each leaf in this tree has a counter of how many times it has been activated.

static add_node_interval_to_stack(low: int, high: int, stack: List)[source]

Updates the stack containing the complement intervals.

The intervals are merged, example:

stack = [] called with (5, 10) -> stack = [5, 10] called with (10, 15) -> stack = [5, 15] called with (18, 20) -> stack = [5, 15, 18, 20]
